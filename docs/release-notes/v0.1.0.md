# codex-zai-proxy v0.1.0

First public release.

## Highlights

- stdio MCP server interface for Codex and other strict MCP clients.
- Streamable HTTP upstream transport for one MCP endpoint per process.
- MCP session header capture and forwarding.
- JSON and SSE response normalization.
- Compatibility handling for empty upstream notification responses.
- Bearer token configuration through process environment or dotenv-style env files.
- Prebuilt GitHub Release binaries for Linux x86_64, macOS Intel, macOS Apple Silicon, and Windows x86_64.

## Install

Download the asset for your platform from this release, verify the matching `.sha256` file, and put the binary on your `PATH` or reference it by absolute path in your MCP client configuration.

See `docs/install.md` for detailed installation instructions.
