# Changelog

## 0.1.0

- Initial stdio-to-Streamable-HTTP MCP compatibility proxy.
- CLI configuration through `--upstream-url`, bearer token options, dotenv-style env files, custom headers, timeouts, and compatibility mode.
- JSON, SSE, empty-response, and MCP session header handling.
- English documentation and Codex configuration examples.
- GitHub Actions CI and release automation.
- Prebuilt release assets for Linux x86_64, macOS Intel, macOS Apple Silicon, and Windows x86_64.
